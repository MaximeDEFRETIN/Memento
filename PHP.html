Le PHP

Le PHP sert à dynamiser un site xeb en permettant de créer un compte, une newsletter, des actualites ...
Avec le PHP, le contenu que l'internaute voit lui est personnel.
C'est un langage côté serveur (présent que sur un serveur) qui génère une page avant quelle soit anvoyée à l'internaute.


Maintenant la programmation

. -> sert à la concaténation.
'' -> affiche un texte.
"" -> permet les variables.
$ -> déclare une variable.
echo -> affiche ce qui le suit.
