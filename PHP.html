Le PHP

Le PHP sert à dynamiser un site web en permettant de créer un compte, une newsletter, des actualites ...
Avec le PHP, le contenu que l'internaute voit lui est personnel.
C'est un langage de programmation orienté serveur (présent que sur un serveur) qui génère une page avant quelle soit anvoyée à l'internaute.
