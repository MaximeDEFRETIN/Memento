L'attribut title (texte alternatif lorsquel'on survol un lien ou une image) et l'attribut alt (information supplémentaire pour les images) rendent le web plus accessible

Dans atom, écrire html et appuyer sur entrée pour que la structure html s'afiche. Idem avec div, ...

Dans atom, appuyer sur ctrl puis cliquer sur des lignes pour sélectionner plusieurs ligne et écrire en même temps.
Permet aussi de sélectionner plusieurs mots dans différentes lignes.

ctrl + maj + m pour redimensionner une page firefox au dimension voulue.

Ne pas faire de sauts de ligne et d'espaces inutiles car les navigateurs les chargent aussi.

Dans le terminal, lancer php fichier.php pour lancer le fichier php.

Pour vérifier les erreurs d'un .php, mettre le nom du fichier Vhost dans le navigateur.

Pas d'echo dans une fonction, en PHP.
